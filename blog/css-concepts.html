Inline vs. Inline-Block

The display property is used to determine how the element is going to relate to other elements. The block property makes it so that the element will stay fixed in its block. When an element is inline, it will never try to take up more space than it needs. It will try to fit into the smallest area possible. Like so: When an element is in a block, it tries to maintain its shape, but if its inline block, it tries to fit into the same line if possible. This makes inline-block useful for making a row of elements.
