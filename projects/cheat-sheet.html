Javascript Cheat Sheet

Syntax What it Does:
for (var i in array) Loop once for each value in the array. i is the index.
.concat(value) Adds the input array or values onto the end of the first array. Returns a new array.
.join(seperator) Joins all the elments in the array into a string, seperated by an optional seperator. If no value iis given, uses commas.
.slice(start,end) Returns a new array starting from the start given and ending at the end of the array or given end.
.reverse() Simply reverses the order of the elements.
.splice(index, amount, item1, item2, ...) Inserts new items into the array at the specified index and with the specified amount.
.shift() Similar to .pop(), removes the first item from an array and returns it.
.reduce(function(total,currentValue, index,arr)) Reduces an array to a single value by applying a function to every element.
